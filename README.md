## 🛠️ 编译步骤

- 下载源代码到本地

```bash
git clone https://github.com/AutoAccountingOrg/AutoAccounting
```
- 使用[Android Studio](https://developer.android.com/studio)打开，等待自动配置完成，如出现失败请配置科学上网

- 点击菜单中 `Build` - `Build Bundle(s) / APK(s)` - `Build APK(s)`
